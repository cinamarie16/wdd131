MY WRITING PROCESS: I'm what's called a pantser, which means that I don't plot out what is going to happen in my books before I write them. It also means that sometimes while I'm writing my characters end up guiding me in different directions than I had originally intended for them.

INSPIRATION: Other than character images, I aso have a Spotify playlist for each book/character.

AUDIOBOOKS: I'm working on creating audio versions. I plan on using AI generated narrators. Because I'm an indie author with limited funds. But also because I like to use technological tools wisely.
